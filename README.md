# Projetos Escolares em PHP

Repositório contendo projetos e atividades desenvolvidos em sala de aula
durante o aprendizado da linguagem PHP.

## Tecnologias utilizadas
- PHP
- HTML
- CSS

## Sobre os projetos
Os projetos deste repositório foram criados com foco em aprendizado,
abordando conceitos básicos como:
- Estrutura de páginas
- Lógica simples
- Condições e formulários

## Observação
Projetos desenvolvidos para fins educacionais.
